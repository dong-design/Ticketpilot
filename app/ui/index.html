提供认证、工单、运行、审批和指标接口。

编排层 Agent 服务

负责分类、优先级判断、工具编排和风险控制。

保存工单、运行记录、工具调用和审批任务。

治理层 审批员 / 管理员闸门

退款等高风险操作必须由高权限角色确认。
